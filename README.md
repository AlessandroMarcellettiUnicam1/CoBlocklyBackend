# CoBlockly Backend

CoBlock is a Domain-Specific Language (DSL) for defining compliance rules in terms of blockchain characterizations, treated as first-class citizens. The main advantage of the proposed DSL is that it allows auditors to focus on blockchain characterizations relevant to DApp auditing with compliance checking, addressing the expressiveness limitations of existing compliance rules languages.

This repository contains the backend service and the core evaluation logic for the **CoBlockly** tool.
